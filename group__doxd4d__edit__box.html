Detailed Description

The D4D_EDIT_BOX object is prepared to be used as an input text object and provide some basic features:

  • Variable position and size
  • Disable and enable capability
  • Handling one line text bigger then is visible area
  • Return carrier and backspace support
  • Cursor capability in keys capture mode
  • pOnUsrMsg pointer to the on-user message callback function
