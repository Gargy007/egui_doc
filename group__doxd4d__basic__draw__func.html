Parameters

psz- pointer on the size structure that defines size of the bevel
color- base color of bevel (the eal used color are computed from this color)
type- type of bevel
radius- round corner radius
Returns
none
Note
Function draws the standard eGUI bevel specified by parameters. The used colors are computed from D4D_BEVEL_FORE_INTENSITY & D4D_BEVEL_BCKG_INTENSITY constants. The logical last draw cursor is not moved.
